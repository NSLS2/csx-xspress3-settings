# csx-xspress3-settings
Calibration for SDD (Silicon Drift Detector) from Rayspec 
This calibration is the result of multiple attempts and approaches. The Soft X-Ray photons' energy is close to the noise comming from the motors moving the instrument, even if they don't move. In order to get this calibration all motors were switched of and the specical ferrite rings from Rayspec were installed on detector signal cables. The calibration was created on Centos 7 server and later moved (coped to RHEL8).
